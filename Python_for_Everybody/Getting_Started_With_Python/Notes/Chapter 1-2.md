# Constants
- fixed values such as numbers, letters, and strings are called 'constants' because their value does not change
- numeric constants are as you expect
- string constants use single quotes or double quotes

    numerical constants
    print(123)

    print(98.6)

    string constant
    print('Hello world')

# Variables
- avoid starting with underscores
- usually lowercase is used
- case sensitive
- bad to start with number and pound signs
- can only use letters and numbers

# Sentences or Lines
x=2 ---> Assignment statement
x=x+2 --> assignment with expression
print(x) ---> print statement

x is a variable
2 is a constant
+ is an operator
print is a function

# Numerical Expressions
+      Addition
-      Subtraction
*      Multiplication
**     Power
%      Remainder
